# 📘 Ansible Learning Repository

This repository is created with the main goal of learning **Ansible** from the ground up.

## 🎯 Purpose

The objective of this project is to build a solid foundation in Ansible, starting from the basics and progressively moving toward more advanced topics.

Rather than jumping directly into complex scenarios, this repository focuses on understanding core concepts step by step.

## 🚀 Learning Path

The content in this repository will evolve over time and may include:

- Basic Ansible concepts  
- Inventory management  
- Ad-hoc commands  
- Playbooks  
- Roles and reusable structures  
- Variables and templating  
- Automation best practices  
- Advanced topics (CI/CD integration, scaling, etc.)

## 📂 Structure

As learning progresses, the repository will be organized into different sections based on topics and complexity levels.

## 🧠 Philosophy

Learning by doing.  
Each example, script, or playbook included here represents hands-on practice and experimentation.

## 📌 Notes

- This is a personal learning project  
- Content may change frequently as knowledge improves  
- Mistakes are part of the process
